今天在学习中文文案排版规范时，我一直在想一个问题：为什么很多内容看起来“没毛病”，却始终不高级。看完这些规范后，答案变得很清楚，问题往往出在细节，而这些细节恰好是运营工作里最容易被忽略的部分。

在做内容运营时，文字本身承载的信息是一层，排版和呈现方式在传递的“信号”是另一层。比如中英文之间是否留空格、专有名词的大小写是否统一、标点是否克制，这些都会直接影响用户对内容专业度的判断。用户未必说得出来哪里不对，但会本能地决定要不要继续看下去。

我以前更关注内容是否有观点、有情绪、有信息量，对规范的敏感度并不高。这次学习让我意识到，规范其实是降低沟通成本的一种方式。像 GitHub、TypeScript 这类名称，如果随意缩写或大小写混乱，会让内容显得临时、随意，也会削弱品牌感和可信度。

还有一个很重要的点是节制感。重复使用感叹号、问号，或者在一句话里堆太多符号，短期看可能更“炸”，长期看会消耗用户的耐心。真正成熟的运营表达，往往是信息清晰、节奏舒服，让人读完没有负担。

整体来看，这些排版规则并不是为了束缚创作，而是帮助内容更稳定地被理解、被信任。对我来说，这次学习更像是在校准自己的内容审美，也是在提醒自己，想把运营这件事做好，细节本身就属于核心能力的一部分。

《NFT、ERC20 vs ERC721、IPFS》

今天主要学习了 NFT 相关的基础概念，包括 ERC20 和 ERC721 的区别，以及 NFT 背后的存储方式。之前我对 NFT 的理解更多停留在“图片”或“收藏品”层面，但通过这节内容，才意识到它本质上是由合约标准和数据结构共同组成的。

ERC20 更偏向同质化资产，所有代币之间没有区别，适合用来做支付或流通；而 ERC721 是非同质化的，每一个 Token 都是独一无二的，这也是 NFT 能够成立的基础。NFT 的“唯一性”并不是图片本身，而是链上 Token ID 和所有权记录。

课程中还讲到了 IPFS 和 Metadata 的作用。NFT 图片通常并不直接存储在链上，而是通过 IPFS 这样的分布式存储保存，链上只记录指向数据的地址和关键信息。Metadata 则负责描述 NFT 的属性，比如名称、图片链接和特征参数。

整体学下来，我感觉 NFT 并不是一个单独的概念，而是合约标准、存储方式和数据指向共同配合的结果。这次学习帮我把很多零散的认知连成了一条完整逻辑，对理解后续 Web3 应用很有帮助。

# 2026-01-14

今天的学习主要围绕 Web3 行业的合规要求、法律风险以及全球监管趋势展开。自 2017 年以来，中国监管部门对虚拟货币相关活动持续收紧，整体基调为“全面封堵金融属性，有限容忍技术创新”。虽然 Web3 作为技术本身并未被禁止，但一旦涉及代币发行、融资、交易或矿池运营等高风险金融场景，便可能面临行政甚至刑事责任。

在核心法律风险方面，代币发行与交易行为风险最高。无论代币被命名为“积分”“凭证”还是“治理 Token”，只要具备融资属性或可流通性，就可能被认定为非法金融活动。即便以“空投”“白名单”等形式分发代币，若实质上面向公众募集资金，仍存在非法吸收公众存款的风险，技术人员参与代币模型设计或合约部署，也可能被认定为共同参与者。

其次是博彩、传销与洗钱风险。部分链游或 NFT 项目存在“充值—抽奖—变现”的闭环机制，容易被认定为开设赌场；而 DAO、挖矿平台中的邀请返利、多级分销模式，一旦涉及团队计酬和层级返佣，也可能构成传销。此外，虚拟货币在场外交易中常被用于规避外汇监管，甚至与电信诈骗、赌博资金相关联，参与者可能在不知情的情况下卷入洗钱链条。

从全球视角看，加密行业正加速走向合规化。欧盟 MiCA 法案已正式生效，美国通过 GENIUS Act、STABLE Act 加强稳定币监管，而 FATF（金融行动特别工作组） 则通过 Travel Rule 要求虚拟资产服务提供商在转账中传递交易双方信息。这些监管措施正在重塑 Web3 行业的发展路径。

总体来看，合规已从“成本”转变为“竞争力”。理解监管逻辑、尊重法律边界，是 Web3 从野蛮生长走向长期发展的必经之路。

《从比特币白皮书到以太坊生态：我对区块链本质的一次理解》

从10月份刚接触btc、bnb以来，一直没有系统地学习过区块链到底是什么，只知道它是一个记录所有交易记录的账本。但在前不久看完比特币白皮书后，让我对区块链有了一个全新的认识，于是，在今天看完手册之后我又去看了一下以太坊的白皮书，发现整个以太坊的形成和生态的发展本质其实也是比特币概念和运行的机制延伸。

如果说比特币白皮书是中本聪一步步回答“钱，为什么不需要银行”这个问题，那么以太坊就是“把代码放到链上执行，看看会发生怎样变化”的角色，并由此组成一个由全球共同维护的“去中心化计算平台”。

但比特币和以太坊也有一些核心区别：btc的功能定位是价值转移，而eth是集价值、规则、应用于一体，btc的可编程性非常有限，因此生态更聚焦于btc本声，而eth原生支持智能合约，因此生态发展更丰富多样，而eth代币则是整个以太坊系统运行的燃料。

其中，eth的代码上链是和btc最本质的区别，因为代码上链能长出各种应用，DeFi、NFT、DAO等等，而由此引申出的L1、L2本质就是解决各项应用的安全性和速度问题，从这个角度来看，以太坊是让整个crypto的生态具备无限可能。

通过学习，我逐渐明白区块链并不只是账户，它更像是一笔一笔的钱的去向记录。每一笔都有来源，都有签名。它没有所谓账户余额的概念，只有交易链条，而私钥，就是花钱的唯一钥匙。所有交易都有一个全球统一时间顺序，用来解决双花问题，每一批交易连在一起，就是区块的雏形。

由此可见，区块链并不是想取代一切，而是为世界提供另一种选择：去中心化，并依靠加密代码和网络共识解决信任问题。而这，也是区块链、crypto最有价值的地方。
