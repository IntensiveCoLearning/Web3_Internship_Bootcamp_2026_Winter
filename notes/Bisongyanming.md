# 一、 两个合规维度

-   形式合规：优化白皮书与合同，严禁“投资回报、保本”等表述，遵循国际披露标准（如 MiCA）。
    
-   深度合规：业务实质去金融化，必须在技术与运营上彻底隔离中国大陆用户，远离募资、洗钱、传销红线。
    

# 二、 从业者核心建议

-   避坑指南：远程入职前需核实牌照真伪，警惕“U支付工资”的法律风险（境内不支持）；优先选择技术/行政等低风险岗位。
    
-   风控自救：建立内部合规审计流程；若遇调查，应根据岗位职责留存合规证据，必要时咨询律师进行资源博弈。
    

# 三、 出金与底线

-   安全出金：首选香港持牌机构或受信任的 OTC，遇到冻卡应如实提供交易证明，利用退赔策略换取账户解冻。
