# 2026-01-18

昨晚8点半开始内部的讨论

发现被卡的点还挺多的

看得出来 大家对定位 有点迷茫 也有人焦虑 这研报该怎么做

毕竟大多都是投研小白

有些细节也拿不准

参与的人也多 大家可能也有不同的想法

尊重各位需求和意见的同时 我是认为 与其把精力和思考放在大家都不擅长或者担心做不好的问题上

不如先把有能力完成的先做好 先完成

毕竟 我们还有一个任务 就是实习营里运营方面的任务

所以 我们需要考虑的是 如何布局以后投入投研这个方向的同时 也可以同时完成运营的任务 然后也学习了运营方面的技巧

一个社区要做起来 可不是一两星期 就能做起来的 何况还要兼顾内容输出

所以 不要有完美思维

先把mvp做出来

然后在我们现有的资源上 撬大杠杆

什么杠杆呢？ 我们现有的平台 ethpanda lxdao 助教们 同学们 我们一起做社区的小伙伴们

后续 应该还有想参合一脚的各路大神大佬们

以我多年的经验 只要有人注意到了 就会有各路神仙和牛鬼蛇神

所以 先把注意力拉我们这里 这应该是首要任务之一

这是我今天晚上想出来的一个小小方案

1.  不做漂亮 高大上 完美的产品 / 内容 停止过度策划
    
2.  把问题抛给大众（questionaire) 他们想看什么 想要什么 有什么痛点 有什么好处 害怕什么 梦想是什么
    
3.  提问最好的几个人 送他们想要的 越多人问越好 激励他们发问
    
4.  同时让他们选出最想要交流/提问的几个大咖
    
5.  给大咖们来个排行榜 邀请大咖们转发给他们投票 （有些大咖可能是排行榜的胜负欲 也有些可能想涨粉 有些纯粹是喜欢分享 交流 吹牛逼 当然还有想卖课的）这些都是social values 这圈子里不缺钱 缺的是“我怕你不知道我牛逼” 或者说 “我就是比谁更牛逼”
    
6.  集合问题 （还有大众需求） 答案 然后做内容输出
    

就暂且写到这里吧

然后整理一些东西 毕竟本懒人堆积了一堆没做的功课和tasks
