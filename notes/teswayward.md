## Web3 安全学习笔记

-   **Web3 安全在保护什么**  
    Web3 安全的核心不是“账号”，而是**私钥和合约逻辑**。一旦私钥泄露或合约有漏洞，资产几乎不可逆地损失。
    
-   **私钥与钱包安全**  
    在 Web3 中，钱包即身份。  
    常见风险包括：
    
    -   私钥/助记词被钓鱼或恶意网站获取
        
    -   签署了自己没看懂的交易
        
    -   使用不安全的浏览器插件或设备  
        本质问题是：**用户是自己的银行**。
        
-   **钓鱼与社会工程攻击**  
    很多攻击不靠技术漏洞，而是靠“骗”：
    
    -   假官网、假空投、假客服
        
    -   恶意签名请求（Approval Drain）
        
    -   仿冒 NFT / 项目方社群  
        Web3 安全中，人往往是最脆弱的一环。
        
-   **智能合约安全**  
    智能合约一旦部署，代码很难修改。  
    常见风险包括：
    
    -   重入攻击（Reentrancy）
        
    -   权限控制错误
        
    -   数学溢出或逻辑缺陷  
        因此合约审计在 Web3 中非常重要。
        
-   **协议与 DeFi 风险**  
    即使合约没漏洞，也可能存在：
    
    -   经济模型被攻击（套利、闪电贷）
        
    -   预言机被操纵
        
    -   流动性不足导致极端价格波动  
        安全不只是代码问题，也是**机制设计问题**。
        
-   **跨链与 Layer 2 风险**  
    跨链桥和 Layer 2 是高风险区域：
    
    -   依赖复杂的验证机制
        
    -   一旦桥被攻破，损失通常巨大  
        历史上很多重大安全事件都来自跨链环节。
        
-   **不可逆性的双刃剑**  
    区块链的“不可篡改”提高了可信度，  
    但也意味着：
    
    -   交易错了很难追回
        
    -   被盗资产恢复成本极高  
        Web3 更强调**事前防护，而非事后追责**。
        
-   **安全的基本原则**  
    一些简单但有效的习惯：
    
    -   助记词永不联网
        
    -   大额资产使用硬件钱包
        
    -   不随意授权、不随意点空投
        
    -   合约交互前尽量了解项目背景

## NFT（Non-Fungible Token）学习笔记

-   **NFT 是什么**  
    NFT 是一种**不可替代的区块链代币**。不可替代的意思是：每一个 NFT 都是独一无二的，不能像比特币那样 1:1 互换。
    
-   **和普通代币的区别**  
    普通代币（如 ETH、USDT）是“同质化”的，彼此没有差别；  
    NFT 更像是**数字世界里的“产权证明”**，强调唯一性和所有权。
    
-   **NFT 存的到底是什么**  
    NFT 本身通常不直接存图片或视频，而是：
    
    -   链上记录：所有者、合约地址、Token ID
        
    -   链下链接：指向图片/音频/元数据（如 IPFS）  
        可以理解为：**链上是证书，内容在链下**。
        
-   **技术基础**  
    大多数 NFT 构建在 **Ethereum** 上，常见标准包括：
    
    -   ERC-721：一对一、完全唯一
        
    -   ERC-1155：支持批量和半同质化资产
        
-   **NFT 能用来做什么**  
    NFT 的价值不只是“图片”：
    
    -   数字艺术与收藏品
        
    -   游戏道具与虚拟资产
        
    -   会员凭证 / 门票
        
    -   IP 授权、品牌权益、身份标识
        
-   **为什么有人愿意为 NFT 付费**  
    核心原因不是文件本身，而是：
    
    -   可验证的所有权
        
    -   可追溯的来源（provenance）
        
    -   稀缺性 + 社群共识  
        NFT 更像是一种**社会与技术共同背书的资产**。
        
-   **争议与问题**  
    NFT 也存在明显争议：
    
    -   价格波动大，投机属性强
        
    -   价值高度依赖叙事与社群
        
    -   版权 ≠ 所有权，容易被误解

-   **以太坊是什么**  
    以太坊是一条“可编程的区块链”。如果说比特币主要是“数字货币”，那以太坊更像是一个**去中心化应用的平台**。
    
-   **核心创新：智能合约**  
    以太坊最大的价值在于**智能合约**。它本质是部署在链上的代码，只要条件满足，就会自动执行，不依赖人工或中介。
    
-   **ETH 的角色**  
    ETH 是以太坊的原生代币，用来：  
    1）支付交易和合约执行的手续费（Gas）  
    2）作为 PoS 机制下的质押资产  
    3）在 DeFi、NFT 等生态中充当基础资产
    
-   **账户体系**  
    以太坊有两种账户：
    
    -   外部账户（EOA）：由私钥控制，人用
        
    -   合约账户：由代码控制，程序用
        
-   **共识机制演进**  
    以太坊已从 PoW 转向 **PoS（权益证明）**，核心目标是：
    
    -   降低能耗
        
    -   提高网络安全性
        
    -   为后续扩容打基础
        
-   **生态系统**  
    以太坊承载了大量应用：
    
    -   DeFi（去中心化金融）
        
    -   NFT
        
    -   DAO
        
    -   Layer 2 扩容方案  
        可以理解为“Web3 的底层操作系统”。
        
-   **现实中的局限**  
    以太坊也有问题，比如：
    
    -   主网交易费在高峰期偏高
        
    -   性能受限，需要 Layer 2 辅助  
        这些问题正通过技术升级逐步缓解。
