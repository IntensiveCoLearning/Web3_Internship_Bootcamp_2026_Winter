**入职前重点检查事项：**

•  项目白皮书（核心逻辑、团队背景、技术可行性、风险披露等）

•  Token 分发机制（总量、释放节奏、锁仓、团队/投资人份额、公平性）

•  收益模型（是否可持续、资金来源、是否依赖新资金流入）

•  是否面向或实际服务中国大陆用户（包括中文宣传、境内推广、用户注册限制等）

•  是否存在返利/拉人头/多级分销结构（任何形式的推荐奖励、层级激励）

•  是否存在保本保收益、固定高收益承诺或其他变相投资承诺

**特别注意：**

Web3 项目常采用“跨境分区”模式规避境内监管风险，通常将敏感业务（如交易、发行、推广）置于境外实体，境内仅负责技术研发或非核心职能。

**重要法律风险提醒：**

中国刑法同时实行**属地管辖**与**属人管辖**原则。即使项目主体在境外实施相关行为，中国刑法仍可对境内参与者行使管辖权。境内个人或实体可能因参与、协助或共同犯罪而承担刑事责任（包括非法集资、非法经营、帮助信息网络犯罪活动罪等）。

此外，中国法律明确规定虚拟货币不具有与法定货币同等的法律地位。以虚拟货币为工具帮助他人实现人民币与外币的非法兑换，属于非法买卖外汇犯罪链条的重要环节，依法应予打击。

建议：入职前务必全面评估上述风险，必要时咨询专业律师，避免因“不知情”或“仅技术支持”而陷入法律困境。

-   区块链说白了就是每条链有多个区块组成。每个区块包括了交易信息和之前的区块摘要。
    
-   优点是安全、透明、不可篡改和快速交易
    
-   区块链的特性：去中心化、真正的不可篡改。
    
-   比特币具有货币属性，价格由市场供需关系决定。交易通过随机生成的钱包地址完成，所有的交易都是公开透明的。
    
-   比特币的缺点；容易被黑产利用。打包速度慢。区块的存储数据有限。
    
-   区块链核心组成部分
    
-   区块链 = 去中心化网络 + 区块链本身（一条不可篡改的分布式账本）
    
-   去中心化网络：由无数节点（矿工）共同维护，通过挖矿验证交易、打包区块
    
-   激励机制：矿工获得区块奖励 + 交易手续费（Gas Fee），用户使用网络需支付代币
    
-   运行流程（经典循环）：
    

1.  用户发起交易

2.  广播全网

3.  节点验证

4.  共识打包成块

5.  上链

6.  发放奖励

-   区块链通过去中心化网络+代币激励，让用户在没有中介的情况下真正拥有数据与资产，但目前仍面临性能、体验与p监管的巨大权衡取舍。 Web3 代表了产权与金融范式的革命，而非仅仅技术升级。
    
-   所谓以太坊是全球共享的去中心化计算机，通过智能合约实现“代码即法律”的可编程区块链平台，被称为区块链2.0代表。其原生代币ETH既是价值载体，也是支付“燃料费（Gas）”的必需品。
    
-   以太坊的目标：成为世界计算机，支持Dapps、DeFi、NFT、DAO等庞大生态。
    
-   以太坊从“高能耗的可编程区块链”成功转型为“低能耗、高扩展、模块化的全球结算层+世界计算机”，目前正处于L2爆发+数据分片前夜的关键阶段，其真正价值在于开启了可编程、无需信任的经济与社会实验新时代。
