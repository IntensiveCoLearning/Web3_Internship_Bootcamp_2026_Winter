# 2026-01-14
<!-- DAILY_CHECKIN_2026-01-14_START -->
参加了web3安全趋势会议

总结如下

### 一、整体数据与损失规模

GoPlus Security 等机构记录显示，2025年Web3生态发生约1200起主要安全事件（B端项目方与C端用户各占一半），总经济损失超过**35亿美元**（部分报告显示全年DeFi/交易所黑客事件累计约20-30亿美元，Lazarus等国家级攻击贡献最大份额）。 相比2024年（约25亿美元损失），2025年攻击更集中于高价值目标，单笔损失屡破纪录。

### 二、两大主要攻击趋势

1.  **精准猎杀（Precision Hunting）**
    
    -   针对高TVL协议、交易所进行长期渗透（国家级黑客如朝鲜Lazarus主导）。
        
    -   典型：**Bybit事件**（2025年2月21日，损失约**15亿美元**，401,000 ETH）。黑客渗透Safe{Wallet}前端开发者，篡改多签界面，导致签名者不知不觉批准恶意交易（供应链攻击+前端劫持）。
        
    -   其他：Lazarus全年盗取超20亿美元，涉及多起交易所冷钱包转移。
        
2.  **广撒网（Broadcasting）**
    
    -   针对普通用户的大规模攻击：钓鱼网站、Rug Pull、貔貅盘、地址投毒。
        
    -   2025年损失占比上升，Deepfake+社会工程学放大效果（单笔损失从数万到数百万美元）。
        

### 三、典型攻击案例（2025年代表性事件）

-   **Bybit**：损失15亿美元（历史最大单笔）。Safe多签前端被篡改，签名者界面显示正常但实际转账到攻击者地址。
    
-   **Sonne Finance**：初始损失约2.23亿美元（Optimism链Compound V2分叉漏洞），但应急响应冻结大部分资金，实际净损失约6000万美元。
    
-   **Balancer**：老牌协议被AI挖掘深层漏洞（2025年11月3日），损失约**1.28亿美元**（V2 Composable Stable Pools rounding error + 批量swap操纵），跨多链（Ethereum、Arbitrum等）。
    
-   **UXLINK**：社会工程学+Deepfake攻击（2025年9月），损失约1100万美元。黑客伪造合作伙伴视频会议，诱导团队成员泄露权限（非Rug Pull，是外部渗透）。
    

### 四、新技术带来的新风险（2025年新兴威胁）

-   **AI + 社会工程学**：Deepfake视频/声音伪造面试官/投资人，诱导下载恶意软件或点击链接（损失超2亿美元）。
    
-   **EIP-7702升级风险**：赋予EOA智能合约能力，黑客设计新型钓鱼，诱导用户签署恶意delegate，导致地址被控自动转走资产（2025年5月后损失超2500万美元，97% delegate用于恶意sweeper）。
    
-   **软件供应链攻击（AI Coding）**：AI辅助编程时代，插件/代码植入后门（如Trust Wallet浏览器扩展2025年12月事件，恶意代码窃取助记词，损失约700-850万美元）。
    
-   **地址投毒（Address Poisoning）**：发送小额尘埃交易，伪造相似地址（首尾相同），诱导用户从历史复制错误地址（2025年单笔最高损失5000万美元USDT）。
    

### 五、安全防范建议（核心“四步大法” + 机构措施）

**C端用户**（普通人防身口诀）：

-   不点：不明链接/私信/会议链接，交叉验证。
    
-   不签：盲签必死，看清签名内容（金额、合约、操作）。
    
-   不装：官方渠道下载App，警惕搜索引擎假冒。
    
-   不转：转账核对完整地址，避免高收益诱饵。
    
-   额外：冷热钱包分离、助记词离线保管。
    

**B端/项目方**：

-   全生命周期安全：开发用OpenZeppelin库、上线前多轮审计。
    
-   防供应链：AI代码/插件人工复核，防止后门。
    
-   应急响应：预设冻结机制、快速止损（Sonne案例成功经验）。
    

### 六、问答精华

-   **判断代币真实性**：用GoPlus Security API、DexTools/GMGN等工具查貔貅/流动性风险。
    
-   **Web3 vs Web2漏洞挖掘区别**：Web3重合约逻辑（重入、溢出、预言机）；Web2重网络/权限；两者都用AI辅助，但Web3更偏经济攻击。
