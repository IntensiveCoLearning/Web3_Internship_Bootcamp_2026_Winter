-   **行业整体格局与市场趋势**：2026 年以太坊可能表现较好，比特币表现可能不佳。比特币更像数字黄金，专注货币应用，面临量子计算挑战，在滞胀年份表现可能不如黄金；以太坊更适合资产发行，虽过去在 L2 路线支持上有问题，但 RWA 带来机会；Solana去年在 MEME 领域炒作后表现一般，需从 Hyperledger 和中心化交易所夺取份额。
    
-   **企业链与新 L1 情况**：新企业链不断涌现，如 tangle 预计会引起轰动但逐渐走下坡路，SQL 的 Ark 链可能无人使用，Romani 户的链与 base 类似，新企业链发展势头强但面临挑战。新的 L1 机会不大，比特币、以太坊和 Cardano 网络效应增强，熊市使区块空间需求减少、建设过度，进入壁垒提高，现有玩家受益，为 L2 构建提供框架的是自然赢家。
    
-   **Web3 发展回顾与展望**：**2025 年回顾**：Web3 上半场靠链上创造和价格上涨的增长模式结束，进入流动性为王阶段。稳定币涨幅超 50%，去中心化交易所发展良好，占比增加，年度有特朗普发meme币、巴菲特投资、政策利好等事件。**2026 年展望**：预测比特币价格在 75,000 - 140,000，传统金融计划提速，稳定币支付进入消费领域，预测市场更标准化专业化，DAT 模式可能退潮，但也面临宏观流动性、监管、安全等风险。
    

-   **Web3 行业发展历程** ：
    
    -   **极客与黄金时代（2009 - 2014 年）** ：只有比特币，是密码学和理想主义者的实验阶段，无智能合约和复杂应用，小圈子讨论去中心化货币理想。
        
    -   **造富与乱象并存阶段（2015 年）** ：以太坊上线，智能合约出现，可在区块链上写代码，爆发 ICO 热潮，去中心化金融（DeFi）发展，第二代中心化交易所崛起。
        
    -   **行业爆发阶段（2020 年起）** ：各种概念轮番爆发，如 NFT、DeFi、GameFi、元宇宙等，数字资产确权，解决前期技术问题，应用层打开，正规军入场。
        
    -   **2024 - 2026 年新趋势** ：特朗普当选后美国对加密行业态度转变，监管松绑，金融管道打通，加密资产进入主流金融资产池，传统金融巨头入场，行业逐渐合规，与现实世界结合。
        
-   **Web3 市场现状** ：
    
    -   **市场规模** ：加密货币总市值 3.1 万亿美元，约为深圳证券交易所股票总市值的一半；2026 年 Web3 市场规模约 69.4 亿美元，年复合增长率 49.8%。
        
    -   **开发者数量** ：2025 年全球月活开发者约 2.5 - 3 万人，人效杠杆率高，核心人才筛选挑剔。
        
    -   **招聘情况** ：2025 年全球主要招聘平台新增约 6.6 万个岗位，反弹 47%，但结构性缺口严重，初级人才过剩，精英极度短缺。
        
    -   **热门岗位竞争** ：初级智能合约开发岗位竞争激烈，因 AI 可部分取代初级开发者；有技术积累、懂特定编程语言和合规法案的人竞争小。
        
    -   **重点增长区域** ：硬核技术板块北美是中心；交易所、资产管理和运营，亚洲的香港和新加坡、中东的阿联酋迪拜是重点区域。
        
-   **Web3 求职渠道** ：
    
    -   **垂直平台** ：如 Smartdeer 的 Web3 求职 APP、Web three Dot career、remote sun 和 Crypto JOBS list 等。
        
    -   **社交网络** ：Founder 会在 Twitter（现 X）发帖招人，招聘信息会整合在 Web3 媒体、项目 DISCORD 官方公告和领英上发布。
        
    -   **链上简历** ：招聘时会看 GitHub 代码提交记录、Snapchat 投票记录、Mirror 上的内容、道治理提案等。
        
-   **Web3 职位画像和薪酬体系** ：
    
    -   **团队特征** ：团队规模不大但多为高精尖人才，业务面向海外用户，公司主体多海外注册，创始人多来自传统互联网圈，有激情、崇尚新技术。
        
    -   **人才偏好** ：偏好 top 院校毕业、有互联网行业背景、做过落地项目的人，看重学习能力和热情，不一定要有区块链经验。
        
    -   **工作环境** ：国际化合作，英文是常用工作语言，不强求坐班，部分大项目会尝试混合办公。
        
    -   **职位比例** ：技术岗占 32% 左右，项目管理（含产品经理）占 27%，其次是运营、市场、社区管理、合规、安全审计和投研等。
        
    -   **薪资情况** ：技术和合规岗起薪高，高级岗位月薪破 10 万可能；运营增长岗若做好增长，奖金和 TOKEN 分成可观；各职位未来有不同发展路径。
        
-   **进入 Web3 的路径和岗位要求** ：
    
    -   **路径** ：从 Web2 迁移，自带资源和方法论，如大厂技术、产品、增长人员，有海外经验者更受欢迎；生态切入，在道里接任务、开源社区贡献代码、投研社区写报告；建立链上声誉，在社交平台输出研报、参加活动。
        
    -   **开发岗要求** ：主流语言是 SOLIDITY，新趋势是 rust 应用增多，其他新语言有 Moo、Carryo 和 FUNC 等，需有安全意识和对 Defi、layer two 的理解。
        
    -   **运营和市场岗要求** ：偏好有社区海外媒体运营经验的人，英文口语能力重要，要懂项目、做过闭环转化和增长任务。
        
    -   **产品和投研岗要求** ：偏好有 Web3 经验、做过积分系统背景的人，有计算机和金融领域知识更有优势。
        
-   **求职技巧建议** ：
    
    -   **保持链上活跃** ：技术人员参与开源代码贡献，非技术人员参与项目治理投票、积累链上徽章。
        
    -   **掌握新语言** ：技术求职者除 SOLIDITY 外，多掌握 rust、move 和 Carol 等语言，选一个方向深钻。
        
    -   **熟用 AI 工具** ：知道用 AI 做什么、能做什么及工具的边界，避免依赖 AI 失去自己的想法。
        
    -   **学会避坑** ：警惕三无团队（匿名、无风险投资、无代码更新），查项目 GitHub 代码提交频率、投资机构背书和核心人员背景。
        
-   **求职要点** ：
    
    -   **劳动合同保障** ：大项目较合规，小项目可能不签正式合同或发加密货币工资，需留意自身权益。
        
    -   **增长与 BD** ：增长是多岗位涉及的概念，BD 带来点对点或大客户增长，增长需学习好的增长机制和积累客户资源。
        
    -   **确认项目方靠谱** ：看官网信息、投资方情况、团队成员背景，早期尽量进大项目。
        
    -   **投研和数据分析** ：核心技能是分析和深度研究能力，需学会用数据分析工具，有代码阅读能力更好；新人要写出有质量的报告，入职后可做行业趋势分析和资金管理。
        
    -   **全职岗位对实习要求** ：全职技术岗位更看重作品和代码库，实习和线上项目经历同样重要。
        
    -   **非技术转码** ：先组建小团队写项目、在开源社区贡献代码，搭建 “电商简历” 后再面试。
        
    -   **展示成果** ：在社交平台写有价值、抓热点的内容，通过点赞、收藏、互动展示。
        
    -   **BD 岗位** ：明确方向后找对应实习岗位，若短期无果可在领域内活跃，积累资源；申请大项目可反复投递对口内容。
