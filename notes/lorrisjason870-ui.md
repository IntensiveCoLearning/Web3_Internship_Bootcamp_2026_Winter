# 2026-01-13
<!-- DAILY_CHECKIN_2026-01-13_START -->
**听了Web3运行原理的分享，以下是一些笔记~**

助记词是一个种子，可以派生出多个私钥

助记词经一定流程（例如先变为一个种子数 seed，然后使用 HD 钱包标准如 BIP-32，BIP-44 等），从这个种子派生（derive）出一个或多个私钥（Private Key）。  
HD 钱包（Hierarchical Deterministic Wallet）允许你用同一个助记词生成多个私钥，每个私钥控制一个或多个区块链地址 /账户。

**私钥（Private Key）：**

本质是一个长字符串，相当于密码

谁拥有私钥，谁就能控制资产

不可重置，丢失只能换钱包

通过私钥可以生成公钥，再生成钱包地址

**助记词（Mnemonic Phrase）：**

由12个或24个单词组成

是为了方便记忆私钥而设计的

一个助记词可以派生出无限个私钥（一对多关系）

通过助记词加派生路径可生成多个账户

**安全原则：**

助记词和私钥绝对不能截图、发送给他人

复制时要小心粘贴板被监控

一旦泄露，资产可能被盗，且无法找回

**Gas Fee 的三层作用：**

-   防垃圾交易淹没网络（提高作恶成本）
    
-   激励打包者/验证者提供算力与带宽
    
-   形成经济闭环：资源使用者付费 → 网络可持续运转
