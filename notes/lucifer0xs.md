# Web3安全与合规笔记

# 一、Web3合规与法律风险

## （一）国内监管要求

核心：金融风险，对技术创新留余地

Web3技术本身不违法，但碰这些易踩雷：发代币/炒币、矿池运营

关键禁令：

-   禁止ICO、虚拟货币交易所，境外平台也不能给国内用户服务
    
-   虚拟货币不能当钱花，没有法定货币地位
    
-   严打用虚拟货币搞非法集资、传销、洗钱、赌博
    

## （二）主要法律风险

1.  **发币/炒币风险**：ICO/IEO/IDO全禁，就算叫“积分”，能融资、流通就可能违法；技术人员参与设计、部署，也算共犯，不能说“只写代码”就免责。
    
2.  **刑事风险（重点防）**：链游有“充值-抽奖-提现”流程，可能算开设赌场；NFT/DAO搞拉人头返利、多级推广，易涉传销；场外换虚拟货币，可能算非法经营，碰黑钱还会涉嫌洗钱。
    
3.  **民事纠纷**：虚拟货币买卖、代人投资的合同基本无效，亏了自己担责；代投要留凭证，别乱扩大投资范围。
    

## （三）全球监管趋势

整体越管越严，各国规则逐渐统一，合规成本变高、门槛提升；部分国家设“监管沙盒”，支持合规创新。

## （四）入职Web3注意点（求职必看）

1.  **雇佣关系**：很多项目在境外注册，国内无合法用工身份，签不了正规合同、不交五险一金，影响落户买房；代签公司能部分解决，但纠纷难维权。
    
2.  **薪酬风险**：“人民币+代币”“全USDT”发工资，不符合劳动法；代币价格波动大可能归零，还会拉低社保缴纳基数。
    
3.  **出金与选项目**：私下换币易因对方黑钱冻卡，尽量留部分人民币工资；入职前查项目，避开拉人头返利、承诺高收益的。
    

## （五）案例启示

-   开虚拟货币赌博平台，技术、客服都要追责；帮人用虚拟货币换汇，算共犯。
    
-   搞“矿机+积分”吸钱、拉人头返利，参与宣传、技术支持都会被追责。
    
-   用虚拟货币洗黑钱，哪怕上游骗子没定罪，自己也可能被判洗钱罪。
    

## （六）其他风险

虚拟货币相关业务均非法，挖矿全面禁止，属于高能耗淘汰产业。

# 二、Web3网络安全防护（学生重点）

## （一）常见骗局/攻击

1.  **钓鱼诈骗**：冒充官方发邮件、加好友，诱骗点链接、泄露钱包私钥/助记词，比如假面试通知、空投福利。
    
2.  **恶意软件**：伪装成面试软件、查重工具，偷换钱包地址、远程控制电脑，盗走资产。
    
3.  **社交套路**：冒充HR、学长，以实习内推、紧急求助为由骗钱、骗信息。
    
4.  **其他**：恶意插件偷换转账地址；弱密码、邮箱被盗，导致所有账号失守。
    

## （二）防护清单

1.  面试只用Zoom、腾讯会议，拒绝陌生专用软件，可疑文件先在虚拟机测试。
    
2.  空投、福利活动，要连钱包、输助记词的全是骗局；主钱包离线保存，用测试钱包参与。
    
3.  软件、插件只从官网/官方商店下，少装无关插件，开启防火墙。
    
4.  重要账号开双重验证（不用短信），每个平台密码不一样，用密码管理器记。
    
5.  转账前核对钱包地址前后几位；私钥/助记词手写保存，绝不截图、传网盘。
    

🔴 应急：疑似被骗先断网，截图留证，找校内信息中心或专业人士帮忙。

# 三、高级骗局提醒

-   假热门文档（如稳定币报告）藏病毒，打开就偷信息；仿冒官方域名（字母造假），迷惑性极强。
    
-   项目方内部人员作恶是大隐患，警惕身边不靠谱的合作者。
    

# 四、实用工具（

-   钱包授权管理：[Revoke.cash](http://Revoke.cash)；钓鱼检测：Scam Sniffer
    
-   密码管理：1Password、Bitwarden；反钓鱼学习：[unphishable.io](http://unphishable.io)
    

# 核心总结

Web3机遇多但风险大，合规上别碰发币、传销、场外黑换汇；安全上牢记防钓鱼、护好私钥、谨慎下载，风险自担，敬畏法律和技术安全。

web3运行原理

钱包、私钥、个人主权

私钥、助记词、地址:三者关系

私钥(PrivateKey):你的"终极签名印章"，谁拥有谁就能控制资产助记词(Seed Phrase):私钥的可读备份(通过派生路径可生成多个账户)助记词和私钥是一对多的地址(Address):公钥截取后的字符串，公开收款号(不是隐私信息，但可能被关联分析)

私钥的安全性

私钥很安全，除非你自己泄漏

私钥是不可重置的密码(丢了就是永远丢)，只能换钱包区块链不认识你是谁，只认识"谁能签名"

大部分资产被盗，不是链被攻破，而是私钥/助记词泄露或授权被钓鱼安全原则

助记词/私钥:不截图、不网盘、不发人、不复制到不可信环境剪贴板是高风险区(很多恶意软件会读剪贴板)任何人一旦拿到私钥/助记词=拿到资产控制权
