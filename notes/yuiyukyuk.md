# 2026-01-12
<!-- DAILY_CHECKIN_2026-01-12_START -->


今天看了一大半Uniswap V2的AMM机制。AMM是个有趣的发明，有点像浮动汇率制，而且V2的模型还是非常简单的线性模型。

**首先，说说其算法。**X\*Y = K。有人讨厌这种Y = K/x的除法，所以两边lg一下，就能成线性和了：lnx+lny = lnz。也就是lnY = lnZ - lnX，好理解、简单，但是好用。

但是**问题也是存在**的，比如当x趋于0或是趋于正无穷，其边际替代率，不那么文邹邹说，也就是价格突变程度，会很大（趋于0时），或小到微不足道（趋于正无穷）。但影响更大的是趋于0时候。即使在它机制中写了最低流动性，但初始流动性池子不大情况下，边际价格变化也会很大。

也就意味着，这种算法机制**需要设定一定的范围才好用**。这种算法机制，也就使得那些meme币，初始价格能涨几千倍。底部塞蛋的人，也就是那些链上科学家，能拿到巨额的超额收益。

**其次是手续费机制。**这笔手续费其实抽取的是双边流动性。买的时候，少给你点币，卖的时候，少给你点usdc或是eth。抽的那部分币给LP了，但是这真能弥补LP的无常损失么？估计要想弄明白，得搭建一个波动率模型，还挺复杂的，有时间再去了解下。

明天去看看价格决定机制，顺便看看以太坊的小知识，今天就这样了。
