# 助记词的延伸脉络：

-   註记词 → 唯一 种子
    
-   种子 → 多个 私钥
    
-   私钥 → 单一公钥
    
-   公钥 → 单一 地址
    

**范例： OKX Web3 钱包只要纪录一个助记词就可以生成很多帐户（Address）**

# 私钥

社会学意义：个人主权的起点

-   密码学提供「可验证的个人主权」
    
-   可以自己生成「权力入口」，无需他人许可
    

最原始的样貌是 256 位元（256 个 0 或 1），相当佔据空间，如果以 16 进位制来表达，可缩短为 64 个字串，其中每一个字串都是由组字 0-9 和大小写英文字母 A-F 随机生成。

# 种子（Seed）

一段随机生成的字串序列，用来生成加密钱包的公私钥对（Keypairs）

一颗种子可以生成多个私钥
