今天发现被休闲黑客松筹备组录取了（）紧急补一下Notion的基础知识，后面再按需现学吧

### 核心三要素：Block（块）、Page（页面）、Database（数据库）

-   Block（块）：Notion 的最小可操作单元，取代传统顶部菜单；光标悬停任意位置点「+」或直接输入斜杠  / ，即可插入文本、待办、标题、表格、图片、视频、音频、网页嵌入、代码块、数学公式、谷歌地图、Figma 图稿、Sketch 文件、PDF 等全部内容；每个块都支持拖拽排序、多选移动、左右分栏及实时调整栏宽。
    
-   Page（页面）：当前所有块组成的界面即为一个页面；页面可无限嵌套——在页面中新建页面，左侧栏自动呈现树状套娃结构，支持点击小三角折叠/展开、拖动重排顺序、甚至跨级拖动形成父子关系；每个页面可独立命名、添加图标（内置或上传）、设置封面图（支持链接自动生成与位置重定位）、开启分享权限（编辑/评论/复制模板），并支持历史版本回溯、评论查看与全屏/字体/导出等深度设置。
    
-   Database（数据库）：以六种视图形态存在——表格（Table）、看板（Board）、画廊（Gallery）、列表（List）、日历（Calendar）、时间线（Timeline），既可作为独立页面，也能以内联（inline）方式嵌入任意页面；每条记录（行）都是一个可点击打开的子页面，支持关联属性（如多选、日期、人员、文件、邮箱、电话等），且所有数据全局打通——任务指派、提醒更新、跨页修改均实时同步至「全部更新」侧边栏与客户端推送。
    

### 斜杠命令与智能格式语法

-   输入  /  后快速键入  todo 、 list 、 image 、 table 、 quote 、 math 、 code  等关键词，通常敲 1–2 字按 Enter 即可生成对应块；
    
-   输入  /callout  创建高亮区块， /@  添加提醒或艾特同事， :  + 英文词触发表情， -  （短横+空格）生成项目符号， 1.  （数字+点+空格）生成序号， ---  自动生成分隔线， #  （井号+空格）生成一级标题， ##   /  ###   生成二/三级标题；
    
-   在文字后输入  /color  快速改主体色， /background color  改块背景色， /turn  切换块类型；
    
-   每个块自带独立链接——点击块前六点 → 「Copy link to block」，即可精准跳转至该块，实现跨页面、跨文档的原子级锚点引用。

# 2026-01-16
<!-- DAILY_CHECKIN_2026-01-16_START -->











今天大概是整合思路，修炼心法的一天。听了太多优秀的分享，难免沉浸在他人的叙事中迷失方向，也是时候把目光收回，直率又坦诚地面对自我了。

回顾本周，焦虑大概是我入营期间最大的阻碍。看着大家陆续有所产出，花样层出不穷，想表达些什么，字符在聊天框里却是删了又删。虽然，大家能够在这里齐聚，本身就带有一种被认可的意味，不少伙伴的鼓励也让我从自卑的阴影中走出。但，总有什么在心中作祟，让我一度畏避。

直到今天在x上读到meme老师的一篇文章，心中的杂念终于烟消云散。

他说，第一份Web3工作，并不会决定你的一生，它更像你第一次进入这个行业时照到自己的一面镜子。正是这样的句子让我产生了好好审视自己一番的念头。至于文章中的一些建议，于我而言几乎是只需无脑照搬的执行准则。

第一， 两年后，你希望在Web3的什么位置？

第二， 你现在的能力，适合在哪种环境成长？

首先，我拥有的只是一些可以应付高考的知识，且目前已退化至半吊子水平。我在大学的半学期，所学习的也不过一些简单的通识课，专业课大概到资本主义经济了，不过实际求职应该没人问我什么是级差地租。其次，我对于岗位的规划，仅仅是基于个人性格及专业背景的排除法，至于排除后的岗位，目前也不具备相应技能。我不会写研报，也不会数据分析，无论在哪个领域，这个假期去苦寻实习，得到的大概也只是毫无含金量的Dirty Work。

如此梳理一番，也平静了不少。当下，学习才是第一要义啊。至于同学们时常给我震撼，我想除去他们本身的优秀之外，还有一个细节我一直忽略了。

不像我，几乎才脱下孔乙己的长衫空手进场，大家来到这里，已经积累了相当丰富的经验，也在爱好中投入了相当多的时间，能从中与世界更好地链接。所以他们很清楚，当下自己要向什么方向进发，由此用目的—手段法进行倒推，自然也能确定什么样的输出最能展现自我。

再回想这些天的学习，有个问题几乎是每天被提起的。

“对于非技术岗位的求职方向，您有什么建议？”

“那要结合你自身的能力和规划才行。”

实际上，这一观点与我今天的顿悟是高度一致的，只是在面对现实时，我时常将其忘却。

其实再往前想，能够走到今天，所有的一切都无非DYOR的反复复现。

从我步入大学的那天起悲观的论调就从未断绝。时常叩问自己，真的就如同众人所说，现在除了考公，别无他法了么？不想随波逐流去做不喜欢的事，也不想逃避一般地选择复读，所以，最终还是决定顺应现状了。只是，相较以前拿出了更多时间探索未来的可能性。我所知道的唯一一件事就是，我所走的是以前从未预想的道路，必须加倍努力，才能在更大的不确定中，再度寻回掌握命运的感觉。

每天数个名词从我脑海中飘过。很庆幸地，我和Web3相遇了，并因此看到一丝希望。我的生活，有在变好。

其实这只是学习生活中极小的插曲，但就像星星姐所说，我们面对的，是这个世界噪声最大的圈子之一。所以我想，锚定自我的课题，未来大概还会出现无数次。只希望那时，自己的心情一如今日。
