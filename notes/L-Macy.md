# 2026-01-13
<!-- DAILY_CHECKIN_2026-01-13_START -->
账户

-   两种：
    
    1.  EOA：人管的，私钥控制。发交易、签合约。
        
    2.  合约账户：代码管的，没私钥。跑智能合约。
        
-   地址：长串数字+字母，从公钥来。私钥超重要，丢了完蛋。
    

钱包

-   工具管私钥，帮签名、连app。
    
-   类型：软件（如MetaMask，便宜但风险）；硬件（如Ledger，安全）；纸钱包（打印，易丢）。
    
-   用时：备份助记词！新手MetaMask起步。
    

交易

-   基本单位：转钱或调合约。
    
-   内容：从哪到哪、多少钱、数据、Gas费。
    
-   Gas：付计算，网络忙贵。
    
-   过程：签→发→上链。Layer2快。
    

场景

-   DeFi：智能合约做金融。Uniswap换币，Aave借钱。Layer2省费，钱包连上玩。风险：合约bug。
    
-   NFT：独特东西，ERC-721。OpenSea买艺术/道具。Layer2如Immutable快。
    
-   Layer2：扩展L1。Optimistic（假设对，有挑战）；ZK（证明快）。DeFi/NFT上用，桥资产过去。

学习eth基础，安装完成所有软件

啥是Ethereum？

-   去中心化区块链平台，不止转账，还能跑代码、建app。
    
-   比比特币牛逼：像全球电脑，支持**智能合约**（自动执行的代码）。
    
-   ETH是它的币，用来付手续费、转钱啥的。
    

核心东西

-   区块链：分布式账本，交易透明，不可改。
    
-   智能合约：代码程序，建dApps、DAO啥的，无中介。
    
-   Gas：每笔操作的费用，用ETH付。网络忙时贵，用Layer2（如Optimism）省钱。
    
-   钱包：存私钥的工具，像MetaMask，用来登录dApps。
    
-   Web3：基于ETH的新互联网，用户控数据。
    

历史简记

-   2013-14：Vitalik Buterin想出来，扩展比特币。
    
-   2015：上线。
    
-   关键事：2016 DAO黑客，分叉出ETC；2022 Merge，转PoS，更环保。
    
-   现在（2026）：生态大，DeFi、NFT、游戏啥都有。
    

怎么学入门

-   先读官方指南：[Ethereum.org](http://Ethereum.org)的“What is Ethereum?”
    
-   搞个钱包：MetaMask，生成地址（私钥别丢！）。
    
-   买点ETH：交易所如Coinbase，小额试水。
    
-   实践：测试网玩合约，Layer2省Gas。
    
-   社区：Reddit、Discord问问题。
    
-   注意：防钓鱼，Gas费波动大。
    

关于L1和L2的区别

-   L1像高速公路主干道，人多车堵；L2像支路或高架，分散流量
    
-   Ethereum用L2来scale（扩展），不改主网就能支持更多用户
