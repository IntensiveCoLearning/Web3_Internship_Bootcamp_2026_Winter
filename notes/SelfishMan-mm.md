怎么做：先厘清公链、协议、DApp、钱包、Token 四层结构，再选一条链（Ethereum/Solana/Polygon）做入口。  
注意点：Token≠股权，合规红线各国不同；国内实习避开发币融资环节。

1.  技能清单  
    是什么：技术岗+非技术岗双轨。  
    怎么做：
    
    -   技术：Solidity/Rust 智能合约、Hardhat/Foundry 框架、Ethers.js 前端交互、 subgraph 索引事件。
        
    -   非技术：Tokenomics 设计、链上数据解析（Dune、Flipside）、社区治理（Discord、Snapshot）。  
        注意点：GitHub 公开贡献是最佳简历；英文读写 > 口语，文档以英文为主。
        
2.  找实习 4 步法  
    是什么：链上足迹 > 冷启动 > 面试 > 合规入职。  
    怎么做：  
    ① 链上足迹：铸造 NFT、提交 PR、在 Gitcoin 捐款，留下可验证地址。  
    ② 冷启动：Twitter+Mirror 关注项目方，DM 简洁三句话：我能解决 X 问题+链上证据+可实习时段。  
    ③ 面试：讲清一个你 debug 过的合约漏洞或数据分析案例。  
    ④ 合规：签三方前确认公司主体所在地，远程合同用 Token 发薪需签两份协议（劳务+Token 奖励）。  
    注意点：DAO 无传统 HR，回复周期 1–7 天；先小额 Grant 试岗再转正。
    
3.  安全守则  
    是什么：私钥即身份，被盗无法挂失。  
    怎么做：硬件钱包+多签+独立浏览器配置；测试网水龙头领水再部署。  
    注意点：永远不要在 README 里放 .env 助记词；Git 历史记录也能泄漏。
    
4.  日常工具箱  
    是什么：链上数据+开发+协作三件套。  
    怎么做：
    
    -   链上：Etherscan、Dune、Nansen。
        
    -   开发：Remix 在线 IDE、Foundry 本地测试、Alchemy 节点。
        
    -   协作：Notion 文档、Snapshot 投票、Safe 多签国库。  
        注意点：主网 Gas 高，优先用二层或测试网做 Demo。
        
5.  成长路径  
    是什么：实习生 → 核心贡献者 → 治理代币持有者。  
    怎么做：每月至少一次链上提案（即使只是更新参数），累积投票权。  
    注意点：DAO 考核看“链上贡献值”，不是打卡时长；离职后仍可凭代币参与治理。
