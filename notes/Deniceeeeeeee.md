# 2026-01-14
<!-- DAILY_CHECKIN_2026-01-14_START -->
今天一连串上了两堂与网络安全和合规相关的课程，对我来说冲击其实挺大的，也让我第一次真正意识到，在 Web3 里，安全并不是进阶选项，而是入门的第一课。正如 Adam 老师所说，当你真正踏入 Web3 和加密货币世界后，你的每一个操作、每一个决定，其实都伴随着风险，只是这些风险往往并不直观，也很容易被忽略。比如在浏览网站需要连接钱包，很多时候我们会下意识地觉得“只是连一下而已，应该没关系”，但往往正是这种放松警惕的瞬间，给了黑客可乘之机。一旦在钓鱼网站上授予了权限，黑客几乎可以为所欲为，钱包里的资产就开始进入倒计时。我也联想到自己曾在钱包里看到一些来历不明的代币，当时没有多加理会，现在回头看反而庆幸自己没有去操作它们，因为这类地址投毒/诱导交易，正是许多资产被盗的起点。只要与陌生地址产生交互，后续就可能被一步步引导，最终造成无法挽回的损失。

通过后续关于安全事件的数据和真实案例分享，我开始更直观地理解 Web3 风险的规模和现实性。成百上千起安全事件、数十亿美元的损失，并不是遥远的行业新闻，而是每天都在发生的事实。从项目方因私钥泄露、多签被渗透而遭遇精准猎杀，到普通用户因钓鱼链接、假代币而被“广撒网”式收割，这些案例让我意识到，去中心化并不意味着没有风险，反而意味着更多安全责任需要由个人承担。在 Web3 世界里，没有客服能帮你追回资产，也没有中心机构为失误兜底，这要求每一个参与者都必须具备更强的独立验证能力和长期的风险意识。更值得警惕的是，随着 AI 的普及，攻击方式也在不断进化，从深度伪造身份到自动化漏洞挖掘，安全攻防已经变成一场持续升级的对抗。

随后的合规课程则让我从“能不能做”转向思考“能不能长期做”。法律的滞后性、不同地区政策的差异，以及现实中大量模糊地带，让 Web3 的每一次业务选择都需要被反复权衡。无论是个人出入金、U 商行为，还是项目融资、市场推广，背后都可能隐藏着合规风险。对我这个新人来说，这些内容让我开始真正思考，在 Web3 里重要的或许不只是如何快速入行，而是如何在这个行业里走得久、走得稳。哪怕现在技术能力和经验还在积累阶段，但至少我开始明白，不点、不签、不装，不转，对每一次授权、转账和合作保持基本的怀疑，才是进入这个世界最现实也最重要的底层能力。

# 2026-01-13
<!-- DAILY_CHECKIN_2026-01-13_START -->

今天布老师关于以太坊运行原理的系统讲解，恰好与我当天早上完成的首次 NFT 铸造实践形成了非常直观的呼应，也让我对早上“在做什么”有了更底层的理解。

早上我按照任务指引，使用指定的 NFT 平台，从基础开始重新梳理了什么是区块链、它的核心特征（去中心化、公开透明、不可篡改）以及基本结构和应用场景。随后进一步学习了NFT 的概念、产品特性与价值来源，也反复被强调 DYOR（Do Your Own Research） 在 Web3 世界中的重要性；链上是公开的，但风险同样也是完全暴露的。

在实操层面，我依次完成了创建钱包（MetaMask / Rabby）、获取 Testnet 代币、连接钱包，并最终成功铸造了一个只属于自己的 NFT。这个过程在当时更多像是“跟着步骤完成任务”，但并未完全理解每一步背后的技术含义。

而布老师从钱包 → 交易 → 出块 → 智能合约执行这一完整路径的讲解，恰好把这些碎片化的操作串成了一条清晰的逻辑链。我开始意识到，钱包并不只是一个“资产容器”，而是私钥管理与签名的入口；一次 NFT 铸造，本质上是一次对合约函数的调用，并最终由网络中的节点验证、打包进区块。

尤其让我印象深刻的是老师带我们阅读智能合约代码和交易细节，以及对节点多样性和去中心化安全性的强调。这让我认识到，Web3 并不是“点击按钮就能赚钱”的黑盒系统，而是一个高度透明、规则写在代码里的系统；是否理解这些规则，决定了你是参与者，还是被动的使用者。

对我个人来说，这一天最大的收获，是从“完成一次链上操作”，转变为开始思考：链上每一个行为，背后究竟发生了什么。这也进一步坚定了我继续深入学习智能合约与链上底层逻辑的方向，而不仅仅停留在表层应用。

# 2026-01-12
<!-- DAILY_CHECKIN_2026-01-12_START -->




从昨天的 Web3 实习计划开营，到今天参加以太坊中文周会、1 小时线上 co-learning 环节，以及 Web3 行业全局介绍和岗位概览，对我这个刚入门不久的新人来说，这些 接触的影响其实还挺大的。

在这短短的时间里，我逐渐对 Web3 有了一个相对清晰的轮廓：从 Web2 和 Web3 的区别开始，到什么是区块链、什么是去中心化，再到智能合约、Web3 的网络安全问题、行业现状、职业机会以及入场方式。这些内容让我意识到，真正跨出的第一步，并不是盲目学习，而是先找到自己的长处，明确自己真正感兴趣的方向，并对相关项目进行基本的调研和理解。

从技术角度来看，区块链是由一个个包含不同信息的区块连接而成，具备公开透明且不可篡改的特性，而这些分布式区块共同组成了一条去中心化的链。比特币和以太坊，正是区块链技术中最成功、也最具代表性的应用。

在去中心化网络中，每一次交易或操作都需要支付费用，也就是所谓的 Gas Fee，以保证交易能够被网络处理和确认。在 Web3 世界里，如果掌握了智能合约语言 Solidity，就可以编写属于自己的合约，定义并执行特定规则。智能合约本质上是存储在区块链上的可执行代码，能够在满足预设条件时自动运行，无需人工干预。

我也逐渐意识到，Solidity 作为一项基础的链上应用技术，对非技术背景的学习者来说确实有一定门槛。但即便如此，我仍然希望能写出自己的第一个智能合约，并完整、独立地跑完从编写、部署到调用的整个流程。
